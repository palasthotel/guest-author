== Description ==

Label users as guest so they cant sign in to your system but you can use them as authors for posts.

== Installation ==

1. Upload `guest-user.zip` to the `/wp-content/plugins/` directory
1. Extract the Plugin to a `guest-user` Folder
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Use checkbox on user edit page
